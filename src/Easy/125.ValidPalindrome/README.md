###基本思路  
	大多数的做法都是一样的：一个指向开头 一个指向结尾 比较两个是否相同  
		- 不相同 直接返回false  
		- 相同 则靠前的指针++、靠后的指针--，直到两者交换相对位置（说明已经超过）  


###函数和用法  
	对于自己处理字符串来说 可以用

```cpp
strcpy(ch, s.c_str());
```

	其中ch是一个申请好了空间的char指针，c_str是字符串s自己的成员函数  
	当然，省去这些也可以直接用下标调用字符串，直接检查s[i]并判断是不是alphanumeric  

	技巧方法  

```cpp
// 判断这个char t是不是字母alpha或数字numeric  
isalnum(t);

// 将t转大写 或小写
toupper(t);
tolower(t);

```
