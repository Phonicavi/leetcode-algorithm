###数字根  
	所有数字的和一直加到只有一位 专有称为 digits root  
	数字根是周期出现的 且具有特点  
		- 所有9的倍数的dr都是9  
		- 两个数和的dr等于这两个数dr的和的dr  

	观察规律  
		num		dr  
		1		1  
		2		2  
		... 	...  
		9		9  
		10		1  
		11		2  
		12		3  
		... 	...  
		17		8  
		18		9  
		19		1  
		... 	...  

	dr(n) = (n-1)%9 + 1;  

```cpp
return (num-1)%9 + 1;
```
